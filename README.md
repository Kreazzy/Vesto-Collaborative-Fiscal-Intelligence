# VESTO | Precision Fiscal Intelligence

**VESTO** is a high-performance, collaborative fiscal tracking ecosystem designed for the modern age. Built by **Cordulatech**, Vesto merges sophisticated financial management with cutting-edge 3D visuals and a neo-fiscal aesthetic.

### 1. Unified Fiscal Dashboard
*   **Real-time Analytics:** Instant breakdown of total balance, income streams, and expenditure.
*   **Live Temporal Tracking:** Integrated clock and calendar system to monitor fiscal windows precisely.
*   **Interactive Controls:** Rapid-entry modals for income and expense logging with categorized allocation.

### 2. Deep-Dive Ledger (Stats)
*   **Adaptive Viewport:** Sophisticated responsiveness that transforms from a high-density table on desktop to an optimized card-stack on mobile devices.
*   **Advanced Filtering:** Search and filter transactions by type, description, or category with zero-latency updates.
*   **Historical Accuracy:** Full audit trail of every transaction with ISO-formatted date tracking.

### 3. User & Identity Management
*   **Secure Authentication:** Hybrid Sign-In/Sign-Up system with persistent local session management.
*   **Custom Identity:** Integrated image processing for profile avatars via local file uploads (Base64).
*   **Preference Control:** Multi-currency support (including custom manual codes) and interface theme toggles (Dark/Light modes).

### 4. Admin Authority Portal
*   **User Directory:** Comprehensive overview of registered accounts.
*   **Moderation Suite:** Admin-level controls for user pruning and system health monitoring.
